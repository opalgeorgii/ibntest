# User Stories

## User Story 1: User Registration
**As a** new user  
**I want to** register an account  
**So that** I can access the platform and save my preferences  

**Acceptance Criteria:**
- User can enter first name, last name, email, and password  
- User receives an error if email already exists  
- Successful registration stores user data and logs them in  

---

## User Story 2: User Login
**As a** registered user  
**I want to** log into my account  
**So that** I can access my personalized content  

**Acceptance Criteria:**
- User can enter email and password  
- System validates credentials  
- User receives error for incorrect login  
- Successful login returns authentication token  

---

## User Story 3: Search Gifts
**As a** user  
**I want to** search for gifts using filters  
**So that** I can find relevant items quickly  

**Acceptance Criteria:**
- User can search by name  
- User can filter by category and condition  
- User can filter by age range  
- Results are displayed dynamically  
- Message shown if no results found  

---

## User Story 4: View Gift Details
**As a** user  
**I want to** view details of a selected gift  
**So that** I can decide if it suits my needs  

**Acceptance Criteria:**
- User can click on a product  
- Product details page is displayed  
- Information includes name, description, and image
